bom_parser_v1.0.0.exe 사용법
1. 프로그램 실행 후 BOM, TEMPLATE, LOOKUPTABLE 파일을 지정 (TEMPLATE, LOOKUPTABLE은 기본적으로 exe파일 같은 폴더내 자동 탐색)
2. OUTPUT 출력을 원하는 폴더를 지정 (BOM 파일 선택시 기본적으로 같은 폴더로 설정됨)
3. RUN 클릭 시 BOM명+"_out"으로 xlsx 파일 생성됨
4. 문제 발생시 txt 파일내에 문제 내역 생성됨(txt 파일 내 [1],[2])

==========================================================
LOOKUPTABLE 파일 관련 수정 방법
1. BOM의 부품이 정상적으로 기입되더라도 정격값들이 누락 될 시 LOOKUPTABLE의 수정이 필요
2. 원하는 정격값들을 입력 시 새로운 행을 생성하여 해당 값을 기입
3. 정격값이 여러개일 시 각 정격값마다 새로운 행을 생성
4. SCHOTTKY DIODE 및 DCDC & LDO에 해당사항이 있을 경우 Subcategory에 해당 내용을 기입하여야 한다.
